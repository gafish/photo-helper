# 照片整理小助手


这是一个用来批量整理照片的小工具，开发这个工具的初衷是为了更好的整理我保存了十几年的海量照片。

## 功能

### v0.1.0

- 按照片拍摄日期将照片一键整理到日期目录
- 通过Hash检测，查找当前目录中的重复照片
- 照片格式过滤，支持视频格式
- 可选是否将整理后的结果保存新建的目录
